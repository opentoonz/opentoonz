## 关于OpenToonz

OpenToonz 是一个由[DWANGO](http://dwango.co.jp/english/)公开坑布的二维动画软件，基于**Toonz Studio 坉坜力版本**， 最早由愝大利公坸[Digital Video, Inc.](http://www.toonz.com/)开坑，并在多年的生产中为[坉坜力工作室](http://www.ghibli.jp/) 定制。
